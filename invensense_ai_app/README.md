# InvenSense AI — Velvet Edition

A deployable multi-page Streamlit prototype for an AI-inspired fast-fashion inventory optimization platform.

## Demo login

- Email: `demo@invensense.ai`
- Password: `demo123`

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Notes

- No database connection.
- No bundled CSV files.
- Uses 50 sample fashion inventory items defined directly in `data.py`.
- Rule-based recommendations only; no OpenAI/API dependency.
